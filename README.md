# 🏥 Health Tracker

A comprehensive Health Tracker web application designed to help users monitor and manage their health records efficiently. The application enables users to track vital health metrics, maintain medical records, monitor progress over time, and gain insights through interactive dashboards and visualizations.

---

## 🚀 Features

- 👤 User Registration & Login
- 🔐 Secure Authentication System
- ❤️ Track Health Metrics
  - Weight
  - Height
  - BMI
  - Blood Pressure
  - Heart Rate
- 📅 Daily Health Record Management
- 📊 Interactive Health Dashboard
- 📈 Progress Tracking and Analytics
- 🔍 View Historical Health Data
- ✏️ Update Health Records
- 🗑️ Delete Health Entries
- 📱 Responsive User Interface

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/your-username/health-tracker.git

cd health-tracker
```

### Install Frontend Dependencies

```bash
cd frontend
npm install
```

### Install Backend Dependencies

```bash
cd ../backend
npm install
```

---

## 🔑 Environment Variables

Create a `.env` file inside the backend folder:

```env
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
```

---

## ▶️ Running the Application

### Start Backend

```bash
npm start
```

or

```bash
nodemon server.js
```

### Start Frontend

```bash
npm run dev
```
